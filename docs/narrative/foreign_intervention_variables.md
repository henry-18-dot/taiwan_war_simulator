# 外国干涉关键变量

本文件用于说明哪些因素会决定外部介入强度，并给系统设计提供映射方向。

| 变量 | 战争意义 | 游戏映射建议 |
|---|---|---|
| 冲突起点 | 封锁、灰区升级与全面登陆会触发完全不同的外部响应。 | 让事件起点直接决定介入等级初值与可用选项。 |
| 前沿基地是否遇袭 | 驻日美军、冲绳、关岛或菲律宾点位遇袭会显著推高介入强度。 | 作为升级阈值，触发盟友授权变化与直接参战概率上升。 |
| 台湾初期存活度 | 台湾若撑住前 72 小时到一周，外援的边际效益会更高。 | 让对手稳定、节点存活和防空状态影响外援转化效率。 |
| 基地与后勤生存性 | 机场、港口、油弹库和跑道抢修决定外军能否持续作战。 | 作为外援吞吐、出动效率与恢复速度的核心修正项。 |
| 盟友国内授权 | 法律约束、政治授权和社会承受度会限制介入形式。 | 用“基地开放度”或“国内授权值”限制高阶支援动作。 |
| 经济与科技冲击 | 航运、芯片、能源与金融震荡会改变各国站队成本。 | 影响制裁强度、外交压力和长期支援持续性。 |
| 核升级风险 | 中美直接交战会压制部分国家的公开打击意愿。 | 让高烈度介入同时抬升全球风险并压缩部分盟友选项。 |
| 其他战区牵制 | 朝鲜半岛、欧洲方向或印度洋压力会分流联盟资源。 | 通过外部牵制事件降低可调用兵力与介入速度。 |

## 设计结论
- 外国干涉更适合建成动态等级系统，而不是单次“是否出兵”判定。
- 介入效果应同时受基地开放度、后勤吞吐、国内授权和升级风险约束。
- 若当前原型继续以中方为玩家，外国干涉最适合映射到国际风险、事件池升级和对手恢复强化。

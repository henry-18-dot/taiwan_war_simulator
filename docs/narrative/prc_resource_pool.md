# 中方资源池

本文件用于把现实背景压缩成游戏可用的能力层，而不是逐单位编制表。

## 前沿打击层
- 火箭军、远程空地打击、远程火箭与察打一体无人机。
- 主要用途是开场瘫痪机场、雷达、港口、通信与指挥节点。
- 游戏含义：高压制、高紧张、高国际风险，适合制造短期节奏优势。

## 海空封控层
- 水面舰队、潜艇、海军航空兵、航母编队、海警、公务船与海上民兵。
- 主要用途是封控航道、压缩空海活动空间、威慑第三方介入。
- 游戏含义：最稳定的持续施压手段，也是灰区到热战之间的核心桥梁。

## 夺岛突击层
- 两栖合成旅、海军陆战队、空降兵、直升机与运输机群。
- 主要用途是夺取滩头、机场、港口和关键节点。
- 游戏含义：高门槛、高战备、高消耗动作，应放在后期危机阶段使用。

## 系统支援层
- 网络战、电子战、卫星侦察、预警机、信息支援与联合后勤。
- 主要用途是争夺信息优势、保障打击链闭合、维持跨军种协同。
- 游戏含义：不直接决定胜负，但会放大其他行动的稳定性与效率。

## 国家后端层
- 军工、造船、无人机产业、铁路、公路、港口、能源储备与民船动员。
- 主要用途是支撑持久封控、补充损耗、扩大运输与装卸能力。
- 游戏含义：决定长期战备恢复与高强度行动的可持续性。

## 设计提醒
- 中方最强之处是“联合封控体系”，不是单一军种。
- 常规两栖运力仍是大规模全面登陆的主要约束。
- 民船、滚装船与港口体系应被视为后续升级的重要资源，而不是背景装饰。
- 无人化平台应作为消耗战与侦察战的关键放大器。

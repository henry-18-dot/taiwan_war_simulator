# 外国干涉资源池

本文件用于把外部介入者的能力压缩成适合游戏建模的支援层，而不是逐国编制表。

## 直接军事介入层
- 以美国为核心，资源包括航母、潜艇、远程轰炸机、前沿海空兵力、太空 ISR、网络战与联合后勤。
- 主要用途是打击两栖输送、压制沿海节点、保住台湾生存空间并维持持续补给。
- 游戏含义：是唯一可能快速改写战役态势的外部力量。

## 前沿基地支撑层
- 以日本、关岛、菲律宾前沿点位为主，涉及机场、港口、油弹补给、维修、疏散与中转设施。
- 主要用途是让外军能前推部署、分散生存并缩短反应时间。
- 游戏含义：决定外援是象征存在，还是能真正形成战役效果。

## 区域盟友作战层
- 日本提供反潜、防空反导、西南岛链火力与基地开放。
- 菲律宾提供北吕宋通道、预置、中转、侦察与撤离支援。
- 澳大利亚提供纵深基地、情报、轰炸机轮换、后场整备与持续后勤。
- 游戏含义：这些国家未必都直接开火，但会显著放大美军效率。

## 次级支援层
- 韩国、英国、加拿大、法国与其他伙伴更可能提供情报、制裁、有限海军存在、工业补给与政治背书。
- 主要用途是释放美军资源、延长联盟支撑链并扩大经济压力。
- 游戏含义：短期难改写前线，长期会显著改变消耗战结构。

## 对华牵制层
- 俄罗斯与朝鲜更现实的作用是战略牵制，而非台海前沿主战。
- 主要用途是在欧洲或半岛制造额外压力，分散美国与盟友注意力和资源。
- 游戏含义：不直接提高台海火力，但会削弱外部干涉速度与规模。

## 非战斗支援层
- 撤侨、搜救、医疗、海运保险、商船调度、港口控制与人道救援体系。
- 主要用途是维持联盟行动合法性、社会承受力与长期运转能力。
- 游戏含义：适合接入外交、制裁、航运与社会韧性事件链。

## 设计提醒
- 外国干涉不是“有没有参战”二选一，而是从侦察、后勤、护航到直接打击的连续光谱。
- 美国是主轴，但日本、菲律宾、澳大利亚决定这条主轴能否高效转起来。
- 欧洲国家更适合建模为政治经济放大器，而不是前沿主战兵力。
- 俄朝更适合作为联盟分压变量，而不是台海地图内的常规援军。

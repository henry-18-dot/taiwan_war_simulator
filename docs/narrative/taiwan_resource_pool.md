# 台湾资源池

本文件用于把台湾防御能力压缩成适合游戏建模的能力层，而不是单纯装备数量表。

## 海峡拒止层
- 岸舰导弹、机动导弹车、火箭炮、空海反舰打击与潜艇。
- 主要用途是破坏海上航渡、压缩登陆编队展开空间、提高渡海成本。
- 游戏含义：决定对手首波攻势能否顺利形成节奏。

## 防空生存层
- 战斗机、防空导弹、雷达网、分散部署、机场抢修与指挥通信冗余。
- 主要用途是保住关键节点、维持有限出动并延长防御体系存活时间。
- 游戏含义：不是追求全面制空，而是避免被开场瘫痪。

## 滩岸守备层
- 陆军、机步、工兵、反甲、便携防空、无人机与城市守备部队。
- 主要用途是拒止抢滩、阻滞空降、守住港口、机场与都会圈要点。
- 游戏含义：是把对手速决战拖入消耗战的核心层。

## 灰区应变层
- 海巡署、军警联合巡逻、近岸监视、海缆与港口防护力量。
- 主要用途是应对越界执法、海上骚扰、基础设施破坏与危机前沿事件。
- 游戏含义：适合放在和平到热战之间的过渡阶段，支撑低烈度回合。

## 全民动员层
- 后备部队、义务役、民防、地方政府、消防、医疗与交通维持体系。
- 主要用途是补充守备兵力、维持社会运转、保护关键基础设施。
- 游戏含义：决定防线能撑多久，也决定城市与后方是否失序。

## 韧性支撑层
- 能源储备、通信骨干、云备份、交通节点、维修补给与外援接收能力。
- 主要用途是保障政府、军队与经济体系在持续打击下仍能运转。
- 游戏含义：是台湾阵营最重要的长期生存资源之一。

## 设计提醒
- 台湾强项不是与对手对称决战，而是把攻击方拖进高损耗、高摩擦环境。
- 海空拒止、后备动员与基础设施韧性应被放在同一套防御体系里。
- 明星装备的价值取决于弹药、维修、分散部署与目标指示链是否还在。
- 海巡、民防与地方应变不应只写成事件背景，它们本身就是战役资源。
